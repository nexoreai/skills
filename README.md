# Nexore Skills

Free, public skills from Nexore (https://nexore.ai). This repo will grow over time.

## What is a skill?
A skill is a small folder with instructions and assets (usually in `SKILL.md`) that tells an AI coding agent how to scaffold or transform a project.

## Example: Optifly
Live site: https://optifly.ai
Live website example built with `saas-landing-template`.

## Included skills
- `saas-landing-template` - Premium light-mode SaaS landing page template (Next.js App Router + TypeScript + Tailwind CSS + shadcn/ui).

## Use `saas-landing-template`
1. Copy the skill into your Codex skills directory (commonly `~/.codex/skills`).
2. Prompt your agent: `Use $saas-landing-template to scaffold a landing page for <your product>.`
3. Or follow the manual steps in `skills/saas-landing-template/SKILL.md` and copy from `skills/saas-landing-template/assets/template`.
